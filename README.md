# openapi-mock-server

- *alpha version*. Creates mock server based on Open API 3.0 specification

## Install

Requires Node 8 or newer.

```bash
$ pnpm i -g openapi-mock-server
$
```

## How to use it?

```bash
$ openapi-mock-server --api=yourapi.yml
OpenApiMock server listening on port 3000
```
